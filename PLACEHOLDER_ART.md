# PLACEHOLDER_ART.md — 占位像素素材说明

当前 `res://art/generated/` 下的 PNG 素材是程序生成的占位像素素材，仅用于验证 v0.2.0 基础玩法闭环。

这些素材不代表最终美术质量，也不能替代后续商业级正式美术。后续可使用 Aseprite、LibreSprite 或 Piskel 按同名文件和尺寸逐步重绘替换。

本项目当前不下载第三方素材；所有占位图均由 `res://tools/generate_pixel_assets.py` 生成，可重复构建和审计。

## v0.2.1 扩展说明

v0.2.1 预备像素素材扩展包仍然是程序生成的扩展素材与占位素材，不代表最终美术质量。

本轮扩展用于系统开发、地图测试、战斗测试、NPC 差异化验证和 UI 布局测试。新增内容包括玩家方向动画、更多 NPC、更多敌人、修仙/末世/赛博地图瓦片、材料/消耗品/武器/特殊物品图标、UI 图标与面板、战斗/魔法特效，以及素材预览图。

后续正式美术建议使用 Aseprite / LibreSprite / Piskel 按相同路径和尺寸逐步重绘替换。当前占位素材禁止作为最终商业美术质量交付。
